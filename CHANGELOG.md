# Changelog

All notable changes to this project will be documented in this file.

## [0.0.0] - 2025-12-30

### Added
- **Settings Page**: Comprehensive configuration area including:
    - **Appearance**: Light/Dark/System theme switching.
    - **Maintenance**: Tool to clean up unused image files from the server.
    - **Backups**: Full system export (JSON + ZIP) and restore functionality with detailed logs.
    - **Profile**: User name/username editing and secure password format.
- **Responsive Design**: Improved mobile navigation for settings with a grid layout.
- **App Icons**: Custom branding applied to application manifest and headers.
- **Database Tools**: Added wipe scripts for testing and development.
